Projeto Ficha de Matriculas foi criado para Gerenciar um banco de dados para controle de matrículas de alunos.
Criação do Banco de Dados
Criação de Tabelas de Alunos - Responsável - Endereço - Telefone - Professores - Histórico de Turma - Histórico Pagamento
Criei View
Criação de Procedures Insert - Update - Delete - Backup e Restore
Criei Triggers de Históricos
Fiz Job - Backup Automatico com Dia e Hora agendado.

Trabelhei com:
DDL (Linguagem de Definição de Dados) - Para Definir a Estrutura - (Create) (Alter) (Drop) (Truncate)
DQl (Liguagem de Consulta de Dados) - Para Consultas Select
DML (Linguagem de Manipulação de Dados) - Manupula os Dados - Insert - Update - Delete
TCL (Linguagem de Controle de Transação) - Gerenciam Transações - Commit - Rollback
Cláusula Where - Having - Group By - Order By - Joins - Top
Operadores And, Or, <, >, <>, =, In, Like
Funções de Agregação Max - Min - AVG - Sum
